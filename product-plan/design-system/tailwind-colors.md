# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue` — Used for buttons, links, active states, and key accents
- **Secondary:** `orange` — Used for secondary actions, highlights, and hover accents
- **Neutral:** `slate` — Used for backgrounds, body text, borders, and structural elements

## Usage Examples

Primary button: `bg-blue-600 hover:bg-blue-700 text-white dark:bg-blue-500 dark:hover:bg-blue-600`
Secondary button: `bg-orange-500 hover:bg-orange-600 text-white`
Active nav item: `bg-blue-50 dark:bg-blue-950 text-blue-700 dark:text-blue-300 border-l-4 border-blue-500`
Neutral text: `text-slate-600 dark:text-slate-400`
Neutral background: `bg-slate-50 dark:bg-slate-950`
Card border: `border border-slate-200 dark:border-slate-800`
